PNG images
==========
The PNG images are available in seven sizes (16x16, 24x24, 32x32, 48x48, 64x64, 128x128, and 256x256). For sizes 32x32 and larger there is a plain version (without shadow) and a shadow version. The shadowless version is only intended for special use cases when a shadow is not desired for any reason. All files have alpha transparency.


ICO files
=========
Each ICO file contains the full set of standard icon sizes which should be provided for full Windows Vista compatibility. Additionally each ICO file contains a 24x24 icon size for displaying the icons in toolbars. Each size is available in three color depths: 32-bit, 8-bit and 4-bit color.
The available icon sizes are:
- 256x256 pixel, 32-bit, 8-bit and 4-bit color, PNG compressed
- 48x48 pixel, 32-bit, 8-bit and 4-bit color, uncompressed
- 32x32 pixel, 32-bit, 8-bit and 4-bit color, uncompressed
- 24x24 pixel, 32-bit, 8-bit and 4-bit color, uncompressed
- 16x16 pixel, 32-bit, 8-bit and 4-bit color, uncompressed
